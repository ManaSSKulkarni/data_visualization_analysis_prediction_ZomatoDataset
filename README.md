# Data Science & Visualization Project  
A complete end-to-end Data Science project involving data cleaning, exploration, feature engineering, visualization, and model building.  
This repository also includes a large dataset (`zomato.csv`) tracked through **Git LFS**.

---

## Project Overview
This project demonstrates the full workflow of a modern Data Science pipeline, including:

- Data Acquisition  
- Data Pre-processing & Cleaning  
- Exploratory Data Analysis (EDA)  
- Visualization (Matplotlib / Seaborn / Plotly)  
- Feature Engineering  
- Model Building & Evaluation  
- Exporting Trained Models (`.pkl`)  
- Insights & Conclusions
